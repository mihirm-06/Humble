<!-- GETTING STARTED -->
## Getting Started

Due to some data constraints, the models must be retrained.

### Prerequisites

Installing packages to train the models: 
* nltk
  ```sh
  pip install --user -U nltk
  ```
* numpy
  ```sh
  pip install --user -U numpy
  ```
